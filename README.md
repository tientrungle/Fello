# Fello
Đây là các file trong 4 tabs Custom code của web Fello trong cài đặt ở Webcake gồm:
fello-body.js là file chứa code trong tab Trước </body>.
fello-head.css là file chứa code trong tab Trước </head>.
fello.css là file chứa code trong tab Custom CSS.
fello.js là file chứa code trong tab Custom JS.
